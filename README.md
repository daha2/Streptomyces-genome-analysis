# Streptomyces-genome-analysis
Example microbial genome analysis workflows using Streptomyces genomes

This repository contains example workflows for microbial genome analysis with a focus on Streptomyces species.

The analyses demonstrate common approaches used in microbial genomics including genome assembly, annotation, biosynthetic gene cluster detection and phylogenomic analysis.

## Analysis workflow

Typical workflow used for bacterial genome analysis:

1. Quality control of sequencing reads  
2. Genome assembly  
3. Genome annotation  
4. Biosynthetic gene cluster detection  
5. Phylogenomic classification  
6. Comparative genomics

## Tools used

- Galaxy platform
- Trim Galore
- FastQC
- Trimmomatic
- Unicycler
- Bowtie2
- Pilon
- FastANI
- Mauve
- BV-BRC
- rastK
- similar genome finder
