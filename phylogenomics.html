Phylogenomics of Streptomyces silvae

Genome-based phylogenetic reconstruction was performed using the TYGS platform and Genome BLAST Distance Phylogeny (GBDP).

Tree reconstruction

The phylogenetic tree was reconstructed using FastME based on GBDP distances calculated from complete genome sequences.

Phylogenetic tree

Figure: Phylogenetic tree based on GBDP distances for the S. silvae dataset. Branch lengths are scaled according to the GBDP distance formula d5. Values at the nodes represent pseudo-bootstrap support values from 100 replicates.
